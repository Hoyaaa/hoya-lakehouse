## 🧑‍💻 Who Am I

> _“데이터 흐름을 이해하고, 그 위에서 인사이트와 의사결정을 설계하는 데이터 분석가”_ 가 되고 싶습니다.

- **최성호 (Hoyaaa)** · **데이터 분석가(志)**  
- 서비스의 **데이터 흐름 전체를 이해**하고,  
  그 위에서 **숫자와 로그를 해석해 인사이트를 뽑아내는 일**을 좋아합니다.
- 필요하면, 분석에 필요한 데이터를 모으거나 검증하기 위해  
  **간단한 앱/웹/스크립트를 직접 만들어 보는 편**입니다.
- 특히 관심 있는 도메인:
  - 학식/캠퍼스처럼 **사람들이 매일 사용하는 공간**
  - **금융/거래 데이터**처럼 숫자와 패턴이 중요한 영역
